# Electronic v1.0.0 — 首个正式发行版

> 个人 AI 助手 · 即时通讯 · 桌面应用

---

## 关于此版本

### 这是 **Electronic** 的第一个正式发行版本（v1.0.0），标志着项目进入稳定可用阶段。

### 集成 AI 大模型、双模式即时通讯（公网/内网）、文件管理与定时任务等核心功能。

---

## 核心功能

### AI 智能对话

- 集成 **LM Studio** 本地大模型（OpenAI 兼容接口）
- **流式输出** 实时显示 AI 回复
- **标记文本渲染** 回复内容，代码块高亮
- **思考内容** 展开/折叠交互
  
  ### 即时通讯
- **双模式聊天**: 公网模式（后端服务器）+ 内网模式（局域网直连）
- **好友管理**: 添加、删除好友，好友申请审批
- **群聊功能**: 可以随意创建、解散、退出群聊
- **图片消息**: 支持发送和查看图片
  
  ### 文件管理
- 文件上传（支持拖拽）
- 文件下载（流式处理与进度显示）
- 文件删除
- localStorage 缓存文件列表
  
  ### 定时任务
- AI 自然语言意图解析（如"15:30 向张三发送 report.docx"）
- 定时发送消息与文件
- 任务持久化与恢复
  
  ### 系统特性
- **暗/亮主题**: CSS 变量驱动，一键切换
- **多语言**: 简体中文 / English / 日本語
- **自动登录**: 30 天凭证有效期
- **运行时动态配置**: 支持运行时修改服务端 API 地址与 AI API 地址（暂不支持使用API Key）

## 构建与安装

支持 **Windows x64** 平台：

- **安装包**: NSIS 安装程序 (`.exe`)，支持自定义安装目录、桌面快捷方式
- 构建命令: `npm run pack`
- 产物输出: `release/Electronic 1.0.0.exe`

---

## 快速开始

```bash
# 安装依赖
npm install
# 浏览器开发模式
npm run dev
# Electron 桌面应用模式
npm run electron
# 构建桌面安装包
npm run pack
```

> *注意: AI 功能需要本地运行大模型，可以手动调整部署计算机IP地址和端口。*
> *通讯功能需要配合后端服务器（默认 127.0.0.1:3001）。*
