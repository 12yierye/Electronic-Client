# Electronic v1.1.0 — 正式版

> 个人 AI 助手 · 即时通讯 · 桌面应用

---

## 关于此版本

这是 **Electronic** 的第二个大版本（v1.1.0），标志着从个人助手向校园级 AI 通讯与内容创作平台的关键跨越。

本版本经历了 4 个 Pre-release 迭代（a→b→c→d），累计新增 **20+ 项功能**，修复 **20+ 个问题**。

---

## 新增功能

### AI Agent 智能引擎

- **Agent 流式编排** — 一句话调度多工具，本地+云端双模推理
- **12 工具注册表** — PPTX/DOCX 生成、图片搜索、文件操作等工具可链式调用
- **追问规划模式** — 执行复杂任务前通过多选提问逐步确认需求（科目→年级→页数→风格）
- **多选题 + 自定义** — 规划卡片支持单选/多选/"其他"输入/"跳过"
- **思考过程可视化** — 可展开/折叠的推理过程展示
- **对话上下文管理** — 历史对话持久化，支持切换与重命名
- **打断/取消** — AbortController 实现流式输出随时中断

### AI 课件生成

- **PPTX 课件** — AI 根据主题自动生成图文并茂的幻灯片
- **DOCX 讲义** — 一键生成课程讲义文档
- **图表/表格/对比** — 多种幻灯片布局自动选择
- **图片自动嵌入** — 联网搜索或 AI 生图，支持多图库（百度/必应/Pixabay/Pexels/Unsplash）
- **LaTeX 公式** — KaTeX 渲染行内/块级公式
- **幻灯片过渡** — fade/push/wipe/dissolve/split 五种动画

### 通讯系统增强

- **组织架构管理** — 学校→班级树形结构，按组织维度查找联系人
- **广播通知** — 面向组织架构的广播消息 + 已读回执追踪
- **E2E 加密（AES-256-GCM）** — 端到端加密保护对话内容
- **离线消息队列** — 断网期间消息自动排队，重连即发
- **用户状态系统** — 在线/忙碌/离线状态切换，头像状态指示
- **消息免打扰** — 按好友/群聊独立设置免打扰

### 内容管理

- **内容库** — 集中管理 AI 生成的课件，支持分类筛选、搜索、分页浏览
- **缩略图预览** — 课件卡片显示首屏缩略图（如无可回退文件类型图标）
- **文件管理** — 上传/下载（流式+进度）/删除

### 系统特性

- **云端模型 API** — 支持 DeepSeek、OpenAI、通义千问、智谱、Moonshot 等多家供应商
- **模型路由器** — 智能路由本地/云端请求
- **三语国际化** — 简体中文 / English / 日本語
- **暗/亮/自动主题** — CSS 变量驱动，跟随系统
- **UI 密度设置** — 紧凑/宽松两种布局
- **发送快捷键** — Enter / Ctrl+Enter 可选
- **导航栏未读角标** — 红点 + 数字，可选闪烁提醒
- **课件模板系统** — 自定义主题色、字体

## 构建与安装

支持 **Windows x64** 平台：

- **安装包**: NSIS 安装程序 (`.exe`)，支持自定义安装目录、桌面快捷方式
- 构建命令: `npm run pack`
- 产物输出: `release/Electronic 1.1.0.exe`

## 升级说明

从 v1.0.x 升级的用户请注意：

1. **配置迁移** — 主题、语言、快捷键、缓存天数等设置自动保留
2. **聊天缓存** — 已读消息 ID 跨版本保持，升级后不会重复提醒
3. **AI 配置** — 本地 LM Studio 配置保留不变；首次使用云端 API 需在设置中填写 Key
4. **内容库** — 之前生成的课件元数据自动识别，无需重新导入
