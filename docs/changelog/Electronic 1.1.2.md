# Electronic 1.1.2

> 个人 AI 助手 · 即时通讯 · 桌面应用

---

## 关于此版本

### 该版本重构了网络架构，实现公网/内网双连接常驻

---

## 新功能

- **公网/内网双连接常驻**：移除全局网络模式切换，公网和内网可同时在线、互不干扰
- **统一身份认证**：基于 JWT token，LAN 服务器通过公网接口验证用户身份
- **统一会话列表**：联系人和群组按来源标记网络（公网/内网），统一展示
- 登录页服务器配置现同时支持公网服务器 URL 和内网 LAN 服务器地址

## 优化

- 所有文件操作（上传、下载、分块传输）支持按网络路由，两端复用同一套服务端 API
- 注册时自动检测公网服务器配置，未配置时弹出引导

## Bug 修复

- 修复了 `electron-updater` 的 CommonJS 模块导入错误
- 修复了登录页服务器配置模态框标签文字空间不足自动换行的问题
- 修复了注册时提示"请先在设置中配置主服务器"但登录页无配置入口的问题
