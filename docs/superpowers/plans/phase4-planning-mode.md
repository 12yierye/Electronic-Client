# Phase 4 — 追问规划模式

> **目标：** AI 收到复杂任务时，先通过多选提问澄清需求，确认完整后再执行

---

## 核心流程

```
用户: "帮我做个课件"
  →
AI: "请问是什么科目？"  [数学] [语文] [英语] [其他]
  → 用户点击 "数学"
AI: "适用于哪个年级？"  [初中] [高一] [高二] [高三]
  → 用户点击 "高二"
AI: "需要几页？"        [8页] [12页] [15页]
  → 用户点击 "12页"
AI: "好的，确认：高二数学课件，12页。开始生成吗？"  [确认] [修改]
  → 用户点击 "确认"
  →
AI: [生成课件...]
```

---

## 实施任务

### 4.1 规划专用 System Prompt

```js
const PLANNING_PROMPT = `你是任务规划助手。用户想完成复杂任务，你必须通过提问来完全理解需求。

规则：
1. 每次只问一个问题，给出 2-5 个具体选项
2. 以 JSON 格式响应，不要有其他内容
3. 当你收集到足够信息后，输出 action: "ready"

输出格式：
{"action":"ask","question":"...","options":["A","B","C"],"context":"第2/5轮"}
或
{"action":"ready","summary":"完整的需求总结","plan":["步骤1","步骤2"]}`
```

### 4.2 前端选择卡片 UI

- AIChat.vue 新增 `v-if="msg.type === 'question'"` 渲染多选按钮
- 用户点击后发送选择到 Agent，追加到对话历史
- 最多 8 轮追问，超时自动执行

### 4.3 智能触发

- 仅部分关键词触发规划模式：`课件|教案|通知|广播|群发|生成`
- 简单对话（如"你好"）直接回答
