# Phase 6 — 教学辅助工具

> **目标：** 扩展为完整的教师工作台

---

## 实施任务

### 6.1 作业批改

- 上传学生作业（图片/文档）
- AI 识别内容 → 对照答案批改 → 给出评分和批注
- 支持语文作文、数学计算、英语写作等科目

### 6.2 试卷生成

- 输入知识点和难度要求
- AI 自动出题（选择题/填空题/解答题）
- 直接导出为 .pptx 或 .docx 试卷

### 6.3 学情分析

- 基于内容库中的作业/考试成绩数据
- 生成学生/班级的成绩分析图
- 薄弱知识点识别 + 针对性练习推荐

### 6.4 教学计划生成

- 输入：学期起止日期 + 教材章节
- 输出：按周的教学计划表（包含课时安排、教学目标、重难点）

### 6.5 家校通知模板

- 预置常用通知模板（家长会、考试通知、活动通知）
- 填空式生成 + 一键广播到目标班级

### 6.6 改动文件

| 文件 | 改动 |
|------|------|
| `toolRegistry.js` | 新增 5+ 教学工具 |
| `handoutGenerator.js` | 试卷/docx 模板 |
| `ContentLibrary.vue` | 按类型（课件/试卷/作业）筛选 |
| `pptxGenerator.js` | 试卷专用模板 |
